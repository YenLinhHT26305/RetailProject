# RetailProject

Data Lakehouse project using

- MinIO
- Apache Spark
- Delta Lake
- Apache Superset

## Architecture

Kaggle Dataset
↓
Python Ingestion
↓
MinIO (Bronze)
↓
Spark (Silver)
↓
Spark (Gold)
↓
Superset Dashboard

## Run

```bash
docker compose up -d
```
